# Demo transcripts

Five varied meeting transcripts for demoing Described. Each is written with speaker labels and timestamps so the AI pipeline extracts clean titles, summaries, action items, key points, and decisions.

| File | Scenario | Good for showing |
|---|---|---|
| `01-product-standup.txt` | Daily product team standup | Action items with clear owners, multiple follow-ups |
| `02-sales-discovery-call.txt` | Sales discovery with a prospect | Decisions + next-step assignments, pricing negotiation |
| `03-design-review.txt` | Design critique of an onboarding flow | Competing opinions, experiment setup, A/B test decisions |
| `04-company-all-hands.txt` | Company all-hands | Big-picture key points, financial numbers, cross-team priorities |
| `05-one-on-one.txt` | Manager 1:1 with an engineer | Career growth conversation, feedback exchange, personal action items |

## How to use

In the web app, click the **Upload** button in the bottom dock, drop one of these files in, and hit **Generate Notes with AI**. Or paste the content into the transcript field.

To demo the Chrome extension path instead, load the extension, open any meeting tab, record, and upload. For a pure transcript demo these files are faster.
